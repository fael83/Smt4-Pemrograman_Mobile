Praktikum pada minggu ke-2 membahas dasar-dasar pemrograman menggunakan bahasa Dart yang meliputi pembuatan variabel serta penggunaan tipe data seperti int, double, String, dan bool. Selain itu juga mempelajari berbagai jenis operator, yaitu operator aritmatika, relasional, logika, assignment, ternary, dan null-aware. Materi ini juga mengenalkan penggunaan final, const, serta konsep null safety untuk menghasilkan program yang lebih aman dan terstruktur. Dalam pelaksanaannya, mahasiswa mengerjakan beberapa studi kasus seperti konversi suhu, kalkulator sederhana, validasi data pengguna, perhitungan diskon, serta penentuan status kelulusan menggunakan struktur Map. Adapun tugas praktikum yang dikerjakan meliputi pembuatan program BMI, konversi mata uang, penerapan seluruh jenis operator dalam satu aplikasi, serta pengembangan aplikasi konversi unit yang dilengkapi menu dan validasi input.
